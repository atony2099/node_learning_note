# 日期处理


## date

```js
new Date().toISOString();
//"2018-09-02T16:18:02.969Z"
new Date().toString();
//"Mon Sep 03 2018 00:19:19 GMT+0800 (China Standard Time)"
```

## timestamp

 A number representing the milliseconds elapsed between 1 January 1970 00:00:00 UTC and the given date.`

```js
new Date().getTime();
Date.now()
```
